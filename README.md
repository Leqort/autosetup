# autosetup

Скрипт для автоматической настройки сервера.

## Быстрый запуск

Выполните на сервере одну команду:

```bash
curl -fsSL https://raw.githubusercontent.com/Leqort/autosetup/main/main.sh -o main.sh && chmod +x main.sh && sudo bash main.sh
```

## Альтернатива через wget

```bash
wget -qO main.sh https://raw.githubusercontent.com/Leqort/autosetup/main/main.sh && chmod +x main.sh && sudo bash main.sh
```
